# Prompt Templates

Templates for AI/ML prompt engineering patterns.

## Expected Files

| File | Purpose | Used By |
|------|---------|---------|
| `classification.md` | Classification prompt patterns | Prompt Engineer |
| `generation.md` | Content generation patterns | Prompt Engineer |
| `extraction.md` | Data extraction patterns | Prompt Engineer |
| `rag-qa.md` | RAG query patterns | Prompt Engineer |

## Status

These templates are created per-project by the Prompt Engineer agent when AI/ML features are being developed.

For projects without AI/ML components, this directory remains empty.
