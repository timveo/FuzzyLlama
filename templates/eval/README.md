# Evaluation Templates

Templates for AI/ML model evaluation and testing.

## Expected Files

| File | Purpose | Used By |
|------|---------|---------|
| `test-dataset.json` | Dataset format template | Model Evaluator |
| `metrics.ts` | Metric implementations | Model Evaluator |
| `comparison-report.md` | Evaluation report template | Model Evaluator |
| `ab-testing.ts` | A/B test framework | Model Evaluator |
| `monitoring.ts` | Production monitoring config | Model Evaluator |

## Status

These templates are created per-project by the Model Evaluator agent when AI/ML features require evaluation.

For projects without AI/ML components, this directory remains empty.
